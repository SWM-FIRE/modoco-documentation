 스프린트 어땠나요

  • 침수때문에 재택을 두번했다.

    • 작업 환경이 열악했다. (인터넷 통신이 잘 안됨)

    • 집가다가 죽을뻔했다.

    • 집에서 물이 샜다.

  • 로그인 생각보다 오래 걸렸다.

    • 작업이 미뤄짐

      • 깃 이슈

    • 주말 이벤트 힘들것 같다.

    • 잔 버그들이 많았다.

    • data fetching 문제

      • useEffect 사용 자제

      • 리팩토링 필요

      • 공부가 필요할 듯

  • 스프린트 분량 적절했다.

  • 스토리 포인트 예상 그래도 예전보다는 나아진거 같다.

    • 아직 3~4시간 오차 있음.

    • 주로 버그나 공부하는 것에서 예상시간 벗어나기도 함.

  • 지라 사용이 조금 저조했다.

    • 기획 관련되어서 그런걸수도

  • 붕뜨는 느낌

    • 이 프로젝트를 왜 시작했는지 다시 상기시키자

    • 싱크 맞추자

    • 중간 발표는 어떤 느낌으로 할건지

    • 우리의 초심

      • 팬 모집

 KPT 회고

Keep - Keep Doing

Problem - Stop doing

Try - Start doing

  • 브랜치 전략 (dev) 세운 것이 좋았다.

  • 집에서 집중이 잘 안된다. 비가 새서ㅜㅜ

  • 푹 자기

  • 문제를 그냥 내버려두지 말고 알아서 소통하면서 해결하려는 모습이 좋았다.

  • 전체 일정이 늦춰졌다.

    • 완벽하게 하려는 마음 때문인지 긴장이 풀려서인지

  • 서로 아는 것도 공유를 하자!

  • 주형님 마감 맞추시려고 노력하는 모습이 모기 좋았다!

  • useEffect문제

    • 리액트 18 공부가 필요하다

  • 다음주 개발은 진짜 필요한 혹은 버그 수정 위주로 하자

  • 서로서로 잘 도와준다.

  • 지라 칸반 보드 활발하지 않았다.

    • 하고 있는 것 넣어주기!

  • 리액트 관련 공부하기 (useEffect)

  • 서로 싱크나 소통이 잘 되는 것 같다.

  • 붕 뜨는 느낌이 들었다.

    • 초심 찾는게 필요할 듯
