이번주 스프린트 어땠나요

Keep

  • Task를 저번보다 더 구체적으로 짰는데 나쁘지 않았다

  • 처음에 들어가기전에, 어떠한 그림이 나올지 설계를 하고 들어가는게 좋았음

  • 백엔드랑 어느정도 소통이 돼서 좋았음 -> sync, 소통 good

  • 1주단위 스프린트 -> 어느정도 안정화 될때까지는 1주단위로 하는게 좋을 거 같음 flush~!

  • emergency 를 그떄그때 합의를 잘 해서 시간분배도 나쁘지 않게 스프린트에 녹여 보냄

  • 일 생기면 잘 알려줬음 [휴가나 일찍 가야되는 상황 같은거를 일주일 전에 말해주니까 계획짜기 수월했다]

  • 항상 새벽에 일어나있어서 너무 좋았음. 새벽러들 개꿀

  • 재택 하루정도는 너무 좋았음

  • 디자이너와의 sync가 잘 맞음

Problem

  • 더 소통해야 된다 -> 지라를 더 활용을 잘 했으면 좋겠다

    • 이슈 기반으로 이야기를 더 해야됨

    • 끝나면 댓글도 좀 달고

    • 멘션으로 체크 해야될것도 좀 알려주고 

  • sync 안맞는 경우

  • 잠이 부족한것 같다.

  • 취미가 하나쯤은 가지고 있자 -> 개발을 벗어나서 마음의 여유를 가질 수 있는 환경 구성

  • 모르는게 넘 많아요 ㅜ

  • CI/CD나 좀 extra 요소(시간을 뺐길 수 있는 모든 요소들)들도 다 고려를 해서 Jira에 issue 할당을 하자

  • 브랜치 전략이 하나 필요하다

  • 소마센터의 문제가 하나둘이 아님

Try

  • 다른사람이 뭘 하고 있는지 파악을 좀 하자

  • 소통이 조금 더 됐으면 좋겠다

  • SSL 안맞았었는데, 저녁에 좀 해결을 하고 갔으면 좋겠음 -> 질문을 최대한 주면 저희가 답안을 잘 해주면 좋겠음

  • 저녁에 한번 sync 맞춰달라고 요청드리면 주형님이 처음부터 열까지 한번 상세하게 알려줘야됨!

  • 기록을 좀더 남겨줬으면 좋겠다

  • 커밋했다! 이정도는 아니고, 

    • ex) 어느정도 보안패치 해결하다가 yarn.lock 자동 수정하는거에 대해서 어떤식으로 해야될지 기타 등등 이런걸 좀 남겨줬으면 다음에 이해가 좋음

  • 브랜치 전략을 세워보자
