정보

외주

  • 외주는 a/s 거의 안 된다

  • 플랫폼이면 유지보수가 어려운 것에 대한 대책이 필요

1. 300만원대면 추천하는 곳

https://www.loud.kr/

  • 디자이너들의 개별 포트폴리오를 볼 수 있음

  • 개별 의뢰 가능

2. 크몽

  • 외주 중개사이트

  • 디자이너 딱 한명을 골라야 함

  • 추천 안 함

    • 차라리 비핸스

    • 실력들이 프로그래머로 비유하면 코더들이 많음

    • 찐실력자는 없음

3. 비핸스

https://www.behance.net/

  • 디자이너들이 포트폴리오 올리는 사이트

  • 각 잡고 올리는 포트폴리오가 많음

    • 레퍼런스용으로 추천

  • 크몽보단 비핸스에서 마음에 드는 디자이너 컨택하는 것 추천

4. 웹 에이전시

단발성 사이트가 아니면 업체 의뢰도 알아봐도 좋을 듯 해

웹 에이전시 := 웹 디자인쪽 외주 일감 전문으로 하는 업체

웹 에이전시 순위 사이트

  • https://www.gdweb.co.kr/main/

계약 시 주의사항 및 팁 (Expert님의 경험담ㅎㅎ)

  1. 처음에는 돈이 큰 상관 없다하더라도 나중에 가면 돌변할 수 있음. 고소한다

    1. 소마가 외주비를 지급해주는데 공휴일(추석)에는 안 해주므로 이 부분 고려해서 계약할 것.

  2. 계약 시 VAT포함 300만원이라고 명시할 것!

  3. 플랫폼이면 단발성인 외주는 별로이다 → A/S에 관해서 최대한 상세하게 계약 할 것

  4. Wireframe을 짜가면 돈이 많이 절약된다

  5. 사실 지인에게 의뢰하는 것이 최고
