진행하면서 받은 피드백

  • 프로필 수정 페이지 리렌더링시 로그아웃 된 것처럼 보임

  • (중요)들어갔던 방 재입장이 가능합니다!!!!!왜 같은방에 한성민이 두 명이 있을 수 있죠??

  • 회원가입 한 후 홈으로 돌아올 때, 맨 아래 화면에서 시작되는 버그가 있습니다. 홈 화면으로 돌아오면 맨 위로 올라갈 수 있도록 해주세요

  • 콘셉트에 따라 노래가 다른 건가요? 방 안에서 콘셉트 변경/어떤 노래인지 간단하게 하단에 적어주면 좋을 것 같습니다.(우주인으로 진행중인데, 좌상단 우주인 누르면 콘셉트 바꾸는 창 나왔으면 좋겠어요)

  • 새로운 채팅 떴을 때, 알림이 뜰 수 있도록 만들면 더 좋을 것 같습니다.

  • 태그 입력할 때, 스페이스바로 태그를 구분하면 대시(-)가 추가되는 것을 확인하였습니다. 이것으로는 제 방의 태그가 붙지 않던데, 태그에 대한 부연설명 추가해주시면 좋을 것 같아요.

  • 아바타 재생성할 때 기존에 들어가있는 방에 자신의 아바타가 갱신되지 않은 것은 의도한 것인가요?

  • 방 삭제는 어떻게 하나요? 만든 사람이 방 터뜨릴 수도 있어야 할 것 같습니다.

  • 방에서 나가기 버튼을 누를 때, 모도코를 나가겠습니까? 가 아니라 이 방을 나가겠습니까?가 더 어울릴 것 같습니다.

  • 초대 기능 넣어주세요

  • 기록 산정 방식이 어떻게 되는건가요? 분 단위로 저장되기에는 그렇게 정교한 로직이 아닌 것 같습니다.

  • 회원가입 후 메인페이지로 보내서 방 생성 찾는게 힘들었다. 회원가입하면 바로 로그인 되게 했으면 좋겠다
